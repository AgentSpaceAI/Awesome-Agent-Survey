# Awesome-Agent-Survey

A curated collection of academic surveys and reviews in the field of AI Agents, including Foundation Models, Multi-Agent Systems, Multimodal Interactions, Agent Design Patterns, GUI-based Agents, Conversational Agents, Game Agents, and more. If you have any relevant papers to contribute, please contact us at: [agentspaceai@gmail.com](mailto:agentspaceai@gmail.com)

This repository is actively maintained and regularly updated with the latest research papers. Stay tuned for more updates! 🚀
